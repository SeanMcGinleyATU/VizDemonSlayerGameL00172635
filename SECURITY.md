## Reporting a Vulnerability

If you discover a vulnerability, please follow these steps:

1. Report the issue directly via the email:
   -L00172635@atu.ie

2. Please include as much as possible about the issue to help us resolve the issue:
   - affected version(s)
   - How to reproduce the issue
   - Any relevant Code snippets and logs pertaining to the issue

3. **Response time:** The response time we aim for when responding to security reports is **2 business days** we aim to provide a resolution or mitigation plan within **5 business days**


---

 ## Disclosure policy
 
 We follow a responsible disclosure process:

  1. We will work with you to help validate and resolve any issue and vulnerability
  2. once the issue has been resolved, we will publicly disclose the details pertaining to the issue or vulnerability

---

## Security best practices

We recommend that all contributors follow these best practices:
 - Use the latest available verison of the project
 - regularly update relevant dependencies
 - Report any suspicious activity or vulnerabilties promptly so they can be dealt with to minimise issues
